# MarbleRoulette
 
그곳에서 난리가 난 `마블룰렛`  
나도 만들어보았다.

## Map
맵은 만들기 힘들어서
1개만 대충 만들어놓았음  
근데 너무 크게 만들었음  

## Player
InputField에 문자열로 추가를 할 수 있도록 만들어놓았다.  
```철수 * 5, 영희 * 10```   
이런 식으로 입력하면 알아서 추가가 될 거임  

셔플 버튼도 만들었음  
누르면 플레이어 정렬 순서가 뒤바뀜  

## Winner
가장 먼저, 가장 나중에 들어온 플레이어가 승리하는 기능 추가를 했다.  
초기 설정 화면에서 고를 수 있음  

## Skill
랜덤한 플레이어에 랜덤한 시간으로 스킬이 활성화 됨  
그냥 주변 플레이어 팅겨냄
